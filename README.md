# StreamList (Week 3)

This project meets Week 3 requirements:
- React app with React Router navigation
- Pages: StreamList (home), Movies, Cart, About
- StreamList logs user input to the browser console
- CSS styling applied
- Free icons via `react-icons`

## Setup (Windows / VS Code)
1. Open a terminal in this folder.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run:
   ```bash
   npm run dev
   ```
4. Open the Local URL shown (usually http://localhost:5173).

## Verify Console Logging
- Go to StreamList page
- Type a title and click Add
- Open DevTools (F12) -> Console
